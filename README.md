星を吸い込んでストックし、状況に応じて「呪文」や「パリィ」を使い分けてボスを討伐する2Dアクションゲームです。


## 🎮 ゲーム概要
* **ジャンル:** 2Dボスバトルアクション
* **プレイ時間:** 約5分
* **コンセプト:** リソース（星）の管理と、状況判断（バフの選択・回避）の楽しさ

## ✨ こだわった実装（技術的なアピールポイント）

本プロジェクトでは、拡張性とゲームフィールの向上を意識してプログラミングを行いました。

* **柔軟なバフシステム（Coroutineの活用）**
  * `Time.timeScale` を変更したUI選択中も正確に時間を計測するため `WaitForSecondsRealtime` を採用。
  * `activeBuffCount` を用いて、複数のバフが重なった際の「点滅演出」が途切れないよう制御。
* **状態遷移（State Pattern）によるキャラクター制御**
  * プレイヤーとボスの行動を `enum`（Normal, Parry, BeamFire等）で管理。
  * `FixedUpdate` 内での物理演算の競合を防ぎ、意図しない挙動（UI操作中の滑りなど）を排除。
* **インスペクターからの調整しやすさ**
  * 呪文のコスト、継続時間、ボスの各アクションの待機時間などをすべて `public` 変数化し、Unityエディタ上からプログラマー以外でもバランス調整ができる設計にしました。

## 🕹️ 操作方法
* **移動:** `←` `→` / `A` `D`
* **ジャンプ:** `↑` / `W` / `Space`
* **吸い込み:** `Z` / `Enter` （長押し）
* **パリィ（すり抜け回避）:** `↓` / `S` （星を1つ消費）
* **呪文メニュー開閉:** `C` （開いている間はスローモーションになります）
